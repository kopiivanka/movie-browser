# Movie Browser

Kotlin + Jetpack Compose Desktop app for browsing movies from TMDB API.  
Implements MVVM architecture with Ktor Client for networking, kotlinx.serialization for JSON, and local JSON storage for favorites.  
Features include movie list, detailed view, favorites management, and persistent local cache.

**Setup:**  
Add `.env` with `TMDB_API_KEY=56219d60a72a4d08566c635421c5ca62` and run `./gradlew run`.

**Stack:** Kotlin · Compose Desktop · MVVM · Ktor Client · kotlinx.serialization · Gradle
